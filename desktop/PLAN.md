## Окно параметров
- задать общие параметры - толщина реза
- задать лист и обрезки - размер листа, сколько отступать от края
- задать детали - размеры, количество название, поворот, кромки

## Окно редактора для десктопа
- делаем три изменяемые по ширине колонки для десктопа
- первая колонка это листы
- вторая это сам раскрой (текущий лист)
- справа детали для распределения на листе

### 1. Колонка переключения между листами
- первая колонка это листы
- кнопка добавить новый лист слева
- по клику переключаемся на новый лист
- удалить лист

### 2. Редактор раскроя
- автоматически при добавлении новой детали выбирается разрез по одной из сторон
- можно поменять разрез нажав на него правой кнопкой
- можно выбрать другой разрез нажав на другую сторону
- можно удалить деталь, нажав правой кнопкой
- можно повернуть деталь двойным кликом если она поворачивается
- пересобрать лист

### 3. Список нераспределенных деталей
- перетаскиванием вставляем деталь в свободную область


## Окно редактора для мобилки
- сверху раскрой
- снизу блок деталей по одной
- перемещение по листам для мобилки в окне раскроя


# Фичи
- поменять ориентацию листа (скорректировать и детали)
- убрать все детали
- выбирать разрез оптимально
- необязательные детали
- детали, которым можно уменьшить высоту или ширину
- несколько кромок
- обрезки
- сколько обрезать от края листа
